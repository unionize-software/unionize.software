This site is designed around data minimization.

## Start Here wizard

The Start Here wizard keeps answers in local application state only. It does not create an account, write to the server, or send answers to analytics.

## Organizer intake

Organizer intake is encrypted in the browser before submission. The server stores ciphertext and limited coarse metadata only.

Encrypted intake is also time-bounded. Rows should expire and be purged on a schedule instead of accumulating indefinitely.

## Analytics

The initial release does not include Google Analytics, session replay, ad pixels, or behavioral tracking.

## Operational caution

Workers should still use personal devices, personal email, and personal phones for organizing activity whenever possible.
