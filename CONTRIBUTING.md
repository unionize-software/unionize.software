# Contributing

## Working rules

- Do not add analytics, session replay, ad pixels, or behavioral tracking.
- Do not connect LLM features to wizard answers or organizer intake data.
- Do not store coworker maps, committee rosters, or campaign CRM data in V1.
- Keep organizing guidance focused on lawful collective action.
- Keep security claims backed by code, tests, and documentation.

## Content changes

Add or update frontmatter fields when editing guides. If a guide changes legal framing, update `last_reviewed` and `review_status`.

## Sensitive information

Never commit private keys, worker intake exports, or real campaign data.
